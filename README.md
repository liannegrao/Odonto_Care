# 📌 Sobre o Projeto

O **OdontoCare** é uma interface moderna e acessível de Login e Cadastro para sistemas odontológicos.

O projeto foi desenvolvido utilizando:

- HTML5
- CSS3
- JavaScript

Com foco em:

- ♿ Acessibilidade
- 🎤 Assistente de voz
- 📱 Responsividade
- ✨ Interface moderna
- 🔐 Experiência intuitiva

---

# 🎤 Assistente de Voz (BETA)

⚠️ O assistente de voz ainda está em fase **BETA**.

Atualmente o sistema já possui:

- Reconhecimento de voz
- Feedback visual
- Navegação assistida
- Controle por microfone

Porém, algumas funcionalidades ainda estão sendo aprimoradas para melhorar:

- Precisão no reconhecimento
- Compatibilidade entre navegadores
- Identificação de e-mails falados
- Estabilidade do sistema
